It can be called by the user any number of times with different points at which the solution is sought. FLINT computes the interpolation coefficients during integration and stores them internally at each natural step of the integrator. During interpolation, these coefficients are used to compute the solutions quickly. This is much less computationally intensive than using the Integrate method
